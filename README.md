# Collocation Practice Studio

A static pink-and-black practice website for learning essay collocations.

## Files

- `index.html` - page structure and essay content
- `styles.css` - visual design
- `script.js` - interactive exercises and answer checking

## Use With GitHub Pages

1. Create a new GitHub repository.
2. Upload these three files to the repository root.
3. Go to `Settings` -> `Pages`.
4. Set the source to `Deploy from a branch`.
5. Choose `main` and `/root`, then save.

GitHub will publish the site after a short wait.
